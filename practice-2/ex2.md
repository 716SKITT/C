Разработать программу, состоящую как минимум из двух пользовательских функций. Программа формирует двоичный код числа в соответствии с количеством байтов, которые отводятся под хранение числа. Если двоичный код названного числа не может быть размещен в выделенную память, то программа дает об этом информационное сообщение. 
Число, количество выделяемой памяти, формат кода (знаковый или беззнаковый) задаются по запросу программы во время ее работы.
Описание особенностей разрядной сетки для размещения кодов целых чисел:
•	Для знаковых чисел старший разряд содержит код знака.
•	Для беззнаковых чисел все разряды отводятся под код числа
