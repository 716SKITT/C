Задание «Статическая очередь»
Задание 1. Реализовать программу, выполняющую стандартный набор 
операций с очередью со сдвигом на основе массива
Задание 2. Реализовать программу, выполняющую стандартный набор 
операций с кольцевой очередью на основе массива.
Стандартный набор действий:
1. проверка пустоты очереди
2. проверка заполненности очереди
3. добавление элемента в конец очереди
4. удаление элемента из начала очереди
5. вывод текущего состояния очереди на экран
Требования к программе:
1. все действия должны быть оформлены как отдельные функции
2. добавлению/удалению должна предшествовать проверка возможности 
выполнения этих операций
3. главная функция реализует следующий набор действий:
 инициализация пустой очереди
 организация диалогового цикла с пользователем
